# titanic-survival-prediction
Built a Titanic Survival Prediction model using Python, Pandas, Scikit-Learn, and Logistic Regression, achieving accurate classification of passenger survival outcomes.
# Titanic Survival Prediction

## Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib

## Steps Performed

1. Data Cleaning
2. Handling Missing Values
3. Feature Selection
4. Logistic Regression Model Training
5. Prediction and Accuracy Evaluation

## Result

The model achieved approximately 75–85% accuracy on the test dataset.
